ベトナムホーチミンの１区にあるDreamPlexというコワーキングスペースに行ってきたのでその詳細、感想をまとめたいと思います！

# ホーチミンのコワーキングスペースDreamPlex
今回行ったのは1区にあるDreamPlexというコワーキングスペースです。

印象はこんな感じでした。

- 現地の人が多め
- とにかく人が多い！
- 設備は可もなく不可もなく
- 丁寧なスタッフ

オフィス街にあり観光客がくるような場所ではなかったため、現地の人が多いのかと思います。

私はベトナム出張中の知り合いの家に泊まったため近かったのですが、普通のノマドワーカーだったらもっと近くて良い場所があるかもしれません。

# 価格
価格はこんな感じでした。

- 1Day: VND 150.000 (約700円)
- 1Week: VND 600.000 (約3000円)
- 1Month: VND 2.000.000 (約9500円)

1日利用700円、1ヶ月で1万円がホーチミンの大体の相場観なので、平均ぐらいといった感じです。

ベトナムの物価を考えると、コワーキングスペースは高めな感じですね。

作業場は机が並びオフィス感のあるところです。

_人が多すぎて私が着いた1時ごろには１席しか空いていません_でした。

１席空いていたのは偶然なので、空いていないなんてことも普通にあると思います。

設備としてはプリントもできるし、キッチン等も自由に使えるようでした。

# 丁寧なスタッフ
女性のスタッフが、設備や料金について流暢な英語で親切に教えてくれました。ちょっと困ってるとすぐに駆け寄って説明してくれたので、スタッフの対応は素晴らしかったです。

# まとめ：ホーチミンのコワーキングスペースDreamPlex
というわけでホーチミン1区のDreamPlexについてまとめてみました。

なかなか行くような場所ではないかもしれませんが、近くに滞在してる方の参考になれば幸いです！
