# 台南の高雄ってどんなところ？
台南の高雄は台北に次ぐ、台湾第二の都市です。

港町のためシーフードが有名で、一年中暖かいためいつ行っても過ごしやすい観光地として有名です。

## 高雄の名付けの親は日本？
高尾と名付けたのは実は日本。

日本の統治時代に、「打狗（ターカウ）」と呼ばれるこの地を日本の音に近い「高雄」に改名したのが始まりになります。

# 台南のオススメ観光地：1.蓮池譚 (龍虎塔など)

<amp-img alt="蓮池譚にある龍虎塔" src="https://storage.googleapis.com/stacky-health/2018/08/26e94c6f025ecf75dd8183ec647e169e-1.jpg" width="1686" height="1264" layout="intrinsic" />

蓮池譚は台南の観光シンボルの１つです。
全長1.4kmの池の周りには20のお寺があり、池の周りをグルっとお寺を渡り歩きながら観光することができます。

特に有名なのは、龍と虎のオブジェが目立つ龍虎塔です。
龍の口から入ることで幸運を招き、虎の口から出ることで厄払いができるそうです。

## 台南のオススメ観光地：2. 六合夜市

<amp-img alt="六合夜市の胡椒餅" src="https://storage.googleapis.com/stacky-health/2018/09/c47f760ceb285d409823ec6da4722e02.jpg" width="1686" height="1264" layout="responsive" />

六合夜市は、全長380mで、約170軒程の屋台が軒を連ねています。
お店とお店の間にテーブルとイスがあり、そこで買った物を食べる事が出来ます。

六合夜市のある高雄は港町のため、シーフードが有名です。
そのためイカ焼き、貝、ロブスターに蟹など、六合夜市には新鮮なシーフードが並んでいます。

台南の夜は、この六合夜市で食べ歩くのが台南観光の定石と言えるでしょう！

## 台南のオススメ観光地：3. 鳥山頭ダム

<amp-img alt="鳥山頭ダム" src="https://storage.googleapis.com/stacky-health/2018/09/1fe78006143d6daaf260fa47a4a6e000.jpg" width="1686" height="1264" layout="intrinsic" />

他に台南の観光地として、鳥山頭ダムがあります。

ここは非日常な景色や体験ができる観光地というよりは社会科見学的なところなので、歴史を知らないと楽しめないかもしれません。

鳥山頭ダムの歴史については記事にまとめているので、そちらを参考にしてみてください。

# 台南のオススメ観光地：まとめ

台南には、「千と千尋の舞台になった九份」のように分かりやすくインパクトのある観光地はないので、正直台北に比べると物足りなさはあります。

とはいえ台南までは台北から新幹線で1時間40分でいけるので、台湾に長期滞在する際には気分転換に来てみるのも良いかもしれませんね。
