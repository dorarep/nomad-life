# 最強カプセルホテル・ファーストキャビンの設備

<amp-img alt="ファーストキャビン阪神西梅田店の外観" src="https://storage.googleapis.com/stacky-health/2018/08/872f6147be26557f9e5ba5bdbefbe6dd.jpg" width="2048" height="1536" layout="intrinsic" />

外観はこんな感じで、値段からは想像できない綺麗さです。

<amp-img alt="ファーストキャビン阪神西梅田店のバー" src="https://storage.googleapis.com/stacky-health/2018/08/b568c9431f79c38367dd4682f95529a9.jpg" width="2048" height="1536" layout="intrinsic" />

中入るとすぐバーが併設されています。

値段も高くなく、カクテルが300円代からでした。

受付の奥には作業スペースがあり、昼間はここで仕事をしている人もいました。

カフェやコワーキングスペースで仕事するようなケースの場合、ホテル自体に作業場があるのはすごいメリットですね。

もちろんWi-Fiも飛んでいます。

<amp-img alt="ファーストキャビン阪神西梅田店の通路" src="https://storage.googleapis.com/stacky-health/2018/11/f4692692a4acb1cf9e3cd347260bddf2.jpg" width="921" height="1228" layout="intrinsic" />


２階が男性用客室で、シックでおしゃれなデザインになっています。

今回は、一番安いビジネスクラスキャビンに宿泊しました。

横両面がすぐ壁で窓等もないため、ベッドに横になったときに「圧迫感があるなー」と感じたのが正直な感想です。

<amp-img alt="ファーストキャビン阪神西梅田店のビジネスクラスキャビン" src="https://storage.googleapis.com/stacky-health/2018/11/df63e3c0c19d2b132b59ead59388cdfa.jpg" width="921" height="1228" layout="intrinsic" />

寝巻きやタオルなど揃っています。

またテレビもついていて、寝ながらテレビを見ることができます。

ただ部屋で音を出してはいけないため、ヘッドフォンで聞く必要があります。

ヘッドフォン自体は貸し出しされていますが、気を使ってあまり音を出せないのは難点ですね。

<amp-img alt="ファーストキャビン阪神西梅田店の貴重品入れ" src="https://storage.googleapis.com/stacky-health/2018/11/461d421cef6187c8d1d473a5809fb443.jpg" width="2048" height="1536" layout="intrinsic" />

客室はカーテンのみで鍵が閉められないため、ベッド横に貴重品をしまう必要があります。

中は広く、バッグも丸々入れることができました。

外出する際もここの鍵を持ち歩く必要があるため、紛失しないように注意しましょう。

また歯磨きやトイレ、風呂等で少し部屋を離れる際も気をつける必要があるのは若干面倒でした。

<amp-img alt="ファーストキャビン阪神西梅田店の操作パネル" src="https://storage.googleapis.com/stacky-health/2018/11/d17ac65e72632203cb16d97952b8a96a.png" width="1632" height="592" layout="intrinsic" />

コンセントや明かりの調整、エアコンの調整もできます。

一見何もなさそうに見えて欲しいと思ったものは揃ってるなーというのが一通り部屋の中を見回してみての感想です。

それ以外でも、必要になりそうなものは一通り貸し出されていました。

<amp-img alt="ファーストキャビン阪神西梅田店の風呂" src="https://storage.googleapis.com/stacky-health/2018/11/aae963f94758e6fdc5e126f3d54832b5.jpg" width="4032" height="3024" layout="intrinsic" />

トイレ、バス、洗濯といった水回りは共用のスペースにあります。

シャンプー、リンス、ボディソープ、綿棒、洗剤などなど揃っており、基本手ぶらで問題なさそうです。

掃除も行き届いていてすごく綺麗です。

<amp-img alt="ファーストキャビン阪神西梅田店のコインランドリー" src="https://storage.googleapis.com/stacky-health/2018/11/11541dfad29dc7a8b141ec383ebd220a.jpg" width="3024" height="4032" layout="intrinsic" />

コインランドリーでは、洗濯300円、洗濯・乾燥が400-500円です。

洗濯できないホテルも多くあるため、コインランドリーがついているのは長期の移動の際に便利ですね。

# ファーストキャビン 阪神西梅田店の価格
私が宿泊した際はAgodaで前日予約して、1泊約4000円でした。

価格帯としてはホテルとゲストハウスの中間といったところで、カプセルホテルとしては妥当といえば妥当なところです。

とはいえ非常に綺麗で必要なものは全て揃っているため、下手なホテルより満足度は高いのではないでしょうか。

ただ寝泊まりするだけであれば、この上ないクオリティだと思いますよ。

# ファーストキャビン 阪神西梅田店の立地
ショップや居酒屋は梅田の方に多くあるのですが、ホテルは大阪駅から梅田から反対方向に15分ほど歩いた位置にあります。

そのため夜に梅田周辺で飲んでから帰るとなると、若干遠いというのが正直な感想です。

最寄駅の福島駅にも商店街や飲食店が多くあるそうなのですが、前情報なしに駅の周りを散歩した感じでは飲食店が多いという印象はありませんでした。

- 寝る場所さえあれば良いけど、個室は欲しいという人
