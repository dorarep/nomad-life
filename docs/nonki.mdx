# 「しろ」が日本一うまい！赤羽のもつ焼き屋のんき
居酒屋好きな先輩が良い店見つけたとのことで、ご一緒させていただくことに。
そのお店は赤羽にある「もつ焼き屋ののんき」です！

平日15時からオープンしているとのことで、開店の15時から予約して突撃してきました！
開店まもない平日15時にも関わらず、店内は人で溢れかえっていました。
（予約は必須ですね）

# もつ焼き屋のんき赤羽店のメニュー
メニューはこんな感じになっています。

## もつ焼き　しろ
最強のメニューはなんといっても「しろ」。

写真をみてわかる通り、口に含むと溶けるような柔らかさで、肉とソースのこってりとした旨味が口いっぱいに広がります…！

<amp-img alt="もつ焼きやのんきの「しろ」" src="https://storage.googleapis.com/stacky-health/2018/09/2015cc20a63ffd5dcbd945cdbf54d92b.jpg" width="1686" height="1264" layout="intrinsic" />

これで100円!?と、同行者と何本も注文してしまいました。
さすがに別格のうまさです！

## ゆで刺身３点盛
生肉に規制が入ってから、代わりに出てきたのが「ゆで肉」ですね！
正直生肉に比べると物足りなさはあるものの、生肉好きにはたまらないうまさでした。

<amp-img alt="もつ焼き屋のんきの「ゆで刺身３点盛」" src="https://storage.googleapis.com/stacky-health/2018/09/73d7b7c6572392ec4497eecc06616dc8.jpg" width="1478" height="1108" layout="intrinsic" />

まずいわけがない、食べ合わせですね！
